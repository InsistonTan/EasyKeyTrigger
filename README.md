# EasyKeyTrigger
简易的键盘按键触发器, 循环触发, 游戏辅助工具

## 简介
本项目基于QT框架(QT6.x)开发, 实现循环触发键盘按键功能, 游戏挂机刷本用

---

## 怎么编译本项目
### 依赖
- QT6.x [QT官网](https://www.qt.io/)
- 开发工具 QT Creator 或者其它
### 编译
- clone或下载本项目源码
- QT Creator打开本项目的 ErChongLuoXuanTool.pro 文件
- 执行qmake
- 编译运行

## 已实现的功能
- 循环触发选择的键盘按键
- 支持设置按键按下持续时间 `也就是按键按下到松开的时间`
- 支持设置循环的间隔时间
- 支持鼠标右键快捷启动/停止
