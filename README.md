# EasyKeyTrigger
简易的键盘按键触发器, 循环触发, 游戏辅助工具

## 简介
本项目基于QT框架(QT6.x)开发, 实现循环触发键盘按键功能, 游戏挂机刷本用

---

## 怎么编译本项目
### 依赖
- QT6.x [QT官网](https://www.qt.io/)
- 开发工具 QT Creator 或者其它
### 编译
- clone或下载本项目源码
- QT Creator打开本项目的 ErChongLuoXuanTool.pro 文件
- 执行qmake
- 编译运行

## 已实现的功能
- 循环触发选择的键盘按键
- 支持设置按键按下持续时间 `也就是按键按下到松开的时间`
- 支持设置循环的间隔时间
- 支持时间随机模式 `开启该模式, 按键按下持续时间和循环的间隔时间 将在一定范围内随机, 避免游戏检测. 时间随机的规则: 按键按下持续时间随机上下浮动五分之一, 例如 150ms, 随机的范围为: 120ms到180ms;
按键触发的间隔时间随机上下浮动十分之一, 例如 5000ms, 随机的范围为: 4500ms到5500ms`
- 支持设置 "启动/停止快捷键". 可设置为快捷键的按键: 鼠标右键, 鼠标侧键1(后退键), 鼠标侧键2(前进键)
